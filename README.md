# Mesto

*Проект посвящен интересным местам.*
Проект включает в себя: 
* Фотографии интересных мест.
* Отметку нравится.
* Возможность изменить профиль.
* Добавление и удаление фотографий.


В ходе выполнения работы использовано:
* HTML5 & CSS3. 
* Native JS
* Адаптивная верстка.
* БЭМ методология
